# UI/UX Design System

## Design Principles
1. **Simplicity First:** Every screen has ONE primary action.
2. **Vet-Friendly:** Large touch targets for iPad use.
3. **Pet-Centric:** Pet photos prominent.
4. **Speed:** Common actions in 1-2 clicks.
5. **Accessible:** WCAG 2.1 AA compliance.

## Color Palette

### Primary (Teal - Medical, Trustworthy)
- `primary-500`: #14b8a6
- `primary-600`: #0d9488

### Accent (Blue - Technology)
- `accent-500`: #3b82f6

### Status Colors
- **Success:** #10b981
- **Warning:** #f59e0b
- **Error:** #ef4444

### Neutrals
- **Background:** #f9fafb (gray-50)
- **Text:** #111827 (gray-900)

## Typography
- **Font Family:** Inter (Google Fonts)
- **Base Size:** 16px
- **Weights:** 400 (Regular), 500 (Medium), 600 (SemiBold), 700 (Bold)

## Component Library

### shadcn/ui Components
- Button, Input, Select, Textarea
- Checkbox, Radio Group
- Date Picker
- Dialog (Modal)
- Alert, Badge
- Card, Table, Tabs

### Custom Components
- **AppointmentCard:** Displays appointment details with status and actions.
- **PetProfile:** Summary card for pet details.
