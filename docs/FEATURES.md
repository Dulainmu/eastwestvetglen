# Feature Specifications

## Phase 1: MVP Features

### 1. Online Booking System
**Goal:** Allow pet owners to book appointments online 24/7.
- **Flow:** Select Service → Select Date/Time → Select Vet (Optional) → Pet Info → Owner Info → Confirm.
- **Key Features:**
  - Real-time availability check.
  - Respects vet schedules and blocked times.
  - Automatic user account creation.
  - Immediate email/SMS confirmation.

### 2. Admin Dashboard
**Goal:** Efficiently manage daily clinic operations.
- **Key Features:**
  - Today's appointment list with status indicators.
  - Quick check-in/check-out actions.
  - Key metrics (Revenue, No-shows).
  - Role-based views (Receptionist vs Vet).

### 3. Automated Reminders
**Goal:** Reduce no-shows with automated communication.
- **Key Features:**
  - Email & SMS reminders sent 24h before appointment.
  - Confirmation links in messages.
  - Delivery logging and status tracking.

### 4. Pet & Owner Records
**Goal:** Comprehensive digital records for better care.
- **Key Features:**
  - Searchable patient database.
  - Medical history timeline.
  - Vaccination tracking with due date alerts.
  - Document attachments.

### 5. Calendar View
**Goal:** Visual schedule management for staff.
- **Key Features:**
  - Day/Week/Month views.
  - Color-coded by service type.
  - Drag-and-drop rescheduling.
  - Vet filtering.
